Machine Learning

One of the greatest technological advancements of the last decade is Machine Learning. Machine Learning is a sub-field of Artificial Intelligence that is defined as computer systems that can mimic human behaviour. Machine Learning is the development of computer systems that can learn by analysing the data provided. In other words, it simply means providing programs with the ability to learn without being programmed explicitly.

Machine Learning Engineers are responsible for creating algorithms that can detect patterns by themselves without being explicitly programmed. The aim of this is to allow systems to learn without human aid and then perform actions accordingly. Machine learning systems search for patterns in data and make improved decisions on future instances.

What Can It Be Used For?

Machine Learning is used to develop AI systems like chat bots, language translation applications, and predictive text. It is also used to develop recommender systems like the ones that recommend products, feeds, and shows for you on Amazon, social media, and Netflix respectively.

Advanced Machine Learning models are also used to develop systems in autonomous vehicles and machines like assisted surgery robots and machines that are used to diagnose medical conditions with images.

Types Of Machine Learning

Supervised Learning

This is the type of machine learning that allows an algorithm to learn patterns from previously labelled data. For example, by providing the system with 200 labelled images of dogs and cats, and then providing a random unlabelled image of a cat, the model will be able to determine that it is a cat.

Unsupervised Learning

This is the type of machine learning that allows an algorithm to learn patterns from unlabelled data. For example, by providing the system with 200 unlabelled images of dogs and cats, and then providing a random unlabelled image of a cat, the model will classify the data into two, the dogs and the cats and any future cat image provided will be automatically grouped with the cats.

Reinforcement Learning

This is a type of machine learning that helps an algorithm learn how to act in a specific environment based on the feedback it receives. It teaches the agent how to act based on rewards and penalties.
